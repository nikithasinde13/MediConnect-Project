✨ Key Highlights

✔ Secure authentication using JWT  
✔ Separate dashboards for Patients & Doctors  
✔ Appointment scheduling made simple  
✔ Digital medical record management  
✔ Clean, scalable full‑stack architecture


---

👥 User Roles

🧑‍⚕️ Doctor

Secure login & profile management

View assigned patients

Manage appointments

Update diagnoses & prescriptions


🧑‍🦱 Patient

Register & login securely

Book doctor appointments

View medical history

Access prescriptions & reports

🚀 Getting Started

🔧 Prerequisites

Java 17+

Node.js & npm

Angular CLI

MySQL


▶️ Backend Setup

cd backend
mvn clean install
mvn spring-boot:run

▶️ Frontend Setup

cd frontend
npm install
ng serve

Access the application at:

http://localhost:4200
